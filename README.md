# Music-Generation
Данная модель предназначена для создания классической музыки. Модель обучена "сочинять" музыку на произведениях только одного Шопена. В качестве начальной последовательности для генерации использованы произведения Моцарта.

Ключевыми инструментами в данном коде являются: **MuseScore 3**, **fluidsynth**, **pyfluidsynth** и **Music21**.

Отличительная черта этого подхода — настройка «температуры». Она регулирует, насколько свободно модель генерирует ответ: чем ниже температура, тем точнее и предсказуемее результат; чем выше — тем больше креатива и неожиданных вариантов.

Особенностью архитектуры модели в том, что она основана на рекуррентных слоях (RNN).
